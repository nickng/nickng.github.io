Go concurrency verification

Golang static analysis process calculi behavioural types

The aim of this line of work is to verify channel-based concurrency interaction in Go using techniques from the research area of process calculi and type systems. This is a joint work with Bernardo Toninho, Julien Lange and Nobuko Yoshida

Tools

nickng/dingo-hunter is a static analyser for Go based on the go/ssa package from the Go project. The tool itself is designed to be an interface to Go programs, performing type inference/model extraction from Go code. Current analysis frontends include:

  • Communicating Finite State Machines (CFSMs) modelling goroutines. This work was presented at Golang UK conference 2016 (video).
  • A mini Go process calculi/type system (MiGo) as a high level, communication-only abstraction. This work was partly presented at Golang UK conference 2017 (video)

Publications

  • Static Deadlock Detection for Concurrent Go by Global Session Graph Synthesis uses the CFSMs synthesis approach.
  • Fencing off Go: Liveness and Safety for Channel-based Programming uses the MiGo approach.
